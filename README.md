## 🗄️ Database

Tabel:
- `users` - data akun
- `heroes` - data hero MLBB
- `items` - data item MLBB
- `teams` - data tim

---

## ⚙️ Cara Install

1. Clone repo ini
2. Import `database/mlbb.sql` ke phpMyAdmin
3. Edit `backend/config/db.php` sesuaikan host/user/pass
4. Jalankan di XAMPP / hosting
